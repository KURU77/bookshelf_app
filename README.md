# 📚 マイ本棚 — バーコード蔵書管理アプリ

本のバーコード（ISBN）をスマホのカメラで読み取って、自分の蔵書を記録・管理するWebアプリです。

## できること

- **バーコードだけで登録** — 本の裏表紙の「978」で始まるバーコードをカメラにかざすだけ。書名・著者・出版社・表紙は自動取得（openBD → Google Books → Amazon書影 → 国会図書館書影の順で検索）
- **置き場所ごとの一覧** — 「一人暮らし先」「実家」「祖父母宅」「研究室」など、本がどこにあるかで分類。置き場所は自由に追加・削除可能
- **表紙のグリッド表示** — 登録した本は表紙画像の一覧で見られる
- **読了 / 未読の管理** — 各本で読了・未読了を切り替え。読書記録があると自動で「読書中」表示
- **読書の記録** — 「いつ・何ページまで読んだか」を記録。総ページ数がわかる本は進捗バーも表示

- **PWA対応** — GitHub Pagesなどで公開すれば、スマホの「ホーム画面に追加」で普通のアプリのように起動できます（オフラインでも一覧の閲覧が可能）

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の構造 |
| `style.css` | デザイン |
| `app.js` | ロジック（スキャン・API検索・保存） |
| `manifest.json` | PWA設定（アプリ名・アイコン） |
| `sw.js` | Service Worker（オフライン対応） |
| `icon-192.png` / `icon-512.png` | アプリアイコン |

データはブラウザの `localStorage` に保存されます（サーバー不要・端末内で完結）。

## 動かし方

### PCで試す

このフォルダで簡易サーバーを起動してブラウザで開きます：

```
python -m http.server 8123
```

→ http://localhost:8123/ を開く

※ カメラでのバーコード読み取りは **HTTPSまたはlocalhost** でのみ動作します（ブラウザの仕様）。PCではISBNの手入力でも登録できます。

### スマホで使う（推奨）— GitHub Pagesで公開

1. GitHubで新しいリポジトリ（例: `bookshelf-app`）を作成
2. このフォルダをプッシュ：
   ```
   git remote add origin https://github.com/<ユーザー名>/bookshelf-app.git
   git push -u origin main
   ```
3. リポジトリの **Settings → Pages** → Branch を `main` / `(root)` にして Save
4. 数分後、スマホで `https://<ユーザー名>.github.io/bookshelf-app/` を開く
5. **ホーム画面に追加**（iPhone: 共有→ホーム画面に追加 / Android: メニュー→アプリをインストール）すればアプリとして使えます

## 注意事項

- データは端末（ブラウザ）ごとに保存されます。別の端末とは共有されません
- ブラウザのデータ消去を行うと蔵書データも消えます
- 書誌検索APIが本を見つけられない場合も、ISBNだけで登録できます
